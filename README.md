# Ark Nova Dashboard

모바일에서도 열기 쉬운 아크노바 시즌 대시보드입니다. 현재 `index.html` 하나로 동작하며, GitHub Pages에 올리면 링크 공유로 바로 접속할 수 있습니다.

## GitHub Pages 배포

1. 이 폴더를 GitHub 저장소에 올립니다.
2. GitHub 저장소의 `Settings`로 이동합니다.
3. `Pages` 메뉴를 엽니다.
4. `Build and deployment`에서 `Source`를 `Deploy from a branch`로 선택합니다.
5. 브랜치를 `main` 또는 사용하는 기본 브랜치로 선택합니다.
6. 폴더는 `/ (root)`를 선택하고 저장합니다.
7. 잠시 뒤 발급되는 GitHub Pages URL을 휴대폰으로 공유하면 됩니다.

## 포함된 파일

- `index.html`: 대시보드 본문
- `.nojekyll`: GitHub Pages에서 정적 파일 그대로 배포되도록 하는 설정
- `aguagu.png`, `boysken.png`, `magicianjang mindbomb.png`: 시즌 7 결과 참고 이미지

## 검색 관련

- 페이지 자체는 검색엔진이 읽기 쉽도록 기본 메타 태그를 넣어두었습니다.
- 공개 저장소로 배포하면 검색엔진에 노출될 수 있지만, 실제 색인까지는 시간이 걸릴 수 있습니다.
- 사이트 내부 검색창은 아직 없어서, 원하면 다음 단계로 추가할 수 있습니다.
